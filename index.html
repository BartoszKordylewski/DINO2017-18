Początek :)

git pull "remote" "branch"

polecenie to merge-uje do naszej gałęzi remote gałąź z serwera o nazwie branch

git branch --merged

Polecenie to służy do sprawdzenia jakie gałęzie zostały połączone z naszą obecną gałęzią

git reset head

Ta komenda pozwala nam na usunięcie pliku z poczekalni w przypadku gdybyśmy dodali tam plik, którego początkowo nieplanowaliśmy. Po tej komendzie po spacji wpisujemy nazwę pliku, którego nie chcemy.

gittk -all

Polecenie służy do wyświetlenia historii wszystkich plików

git checkout "branch"

Komenda ta pozwala nam się przełączyć na gałąź o nazwie branch

git config --global user.name ""

Polecenie to ustawia nam nazwę użytkownika. W cudzysłowia wstawiamy nazwę użytkownika, pozostawiając je pustę resetujemy nazwę użytkownika. Zmienna --global służy do tego, by zapamiętało wpisaną przez nas nazwę użytkownika na stałe, a nie trzeba było wpisywać jej co każdą kolejną sesje.

git pull "remote" "branch"
